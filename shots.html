How it works - Create a free account

  1. Create a TwilioÂ® account and get a phone number (Click Here)
  2. In Twilio, set the Webhook URL for your phone number to "https://evodialer.com/basic.php"
  3. In Twilite Click on the "Programs" drop down menu at the top of the page and select "New Program"
  4. Give the program a name. (The name should be the phone number from Twilio)
  5. Write some Twilite BASIC code. Maybe just something like  say "Hello World" 
  6. Press the Save button and Call the phone number!
